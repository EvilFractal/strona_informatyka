Aby uzyskać listę wszystkich czynników pierwszych liczby naturalnej n, należy dla każdej liczby pierwszej mniejszej od n sprawdzić, czy liczba n jest przez nią podzielna - jeśli jest, należy dzielić liczbę n przez ten czynnik dopóty, dopóki liczba n będzie przez niego podzielna, odnotowując jednocześnie każde tego czynnika wystąpienie.
Funkcja fact(n) rozkłada liczbę n na czynniki pierwsze. Z racji tego, że parzysta jest aż co druga liczba naturalna, wyeliminowanie wszystkich czynników postaci 2 odbywa się w osobnej pętli, po czym następna rozpatruje już tylko liczby nieparzyste.
Na koniec funkcja fact(n) wypisuje - jeżeli takowy pozostał, a więc jest on większy od pierwiastka z n - ostatni czynnik. 

void fact(int n){ 
    int i=2;
    while(n%i==0){
        cout<<i;
        n/=i;
        if(n!=1){
            cout<<'*';
        }
    }
    i=3;
    while(i*i<=n){
        if(n%i==0){
            cout<<i;
            n/=i;
            if(n!=1){
                cout<<'*';
            }
        }
        else{
            i+=2;
        }
    }
    if(n!=1) cout<<n;
}

Warto zauważyć, że powyższa funkcja nie sprawdza pierwszości kolejnych wartości i. Nie jest to potrzebne, ponieważ pętla przechodząca przez kolejne nieparzyste liczby naturalne (jedyną parzystą liczbą pierwszą jest 2) gwarantuje wcześniejsze wyeliminowanie czynników niebędącego liczbą pierwszą i, dzięki czemu nowe n - nieposiadające już tych czynników - na pewno nie będzie przez liczbę złożoną i podzielne.
