wyszukiwanie binarne

Algorytm wyszukiwania binarnego służy do szybkiego znajdowania indeksu szukanego elementu na posortowanej liście poprzez stopniowe zawężanie przedziału, w którym element ten może się znajdować. Aby to osiągnąć, funkcja binary_search(v,n) szukająca na liście v liczby n za pomocą pętli dla środka s każdego kolejnego - coraz mniejszego, do momentu, gdy przedział zawierał będzie tylko jeden element - rozpatrywanego przedziału [poc, kon) sprawdza, czy v[s] jest mniejsze od n - jeśli jest, n na pewno nie znajduje się na indeksie nie większym niż s. W przeciwnym przypadku pierwsze wystąpienie liczby n na pewno nie znajduje się dalej niż s. 

int binary_search(vector<int> v, int n){
    int poc=0,s,kon=v.size();
    while(poc<kon){
        s=(poc+kon+1)/2;
        if(v[s]<n){
            poc=s+1;
        }
        else{
            kon=s;
        }
    }
    return kon;
}

Warto zauważyć, że w przypadku, gdy lista v może liczby n nie zawierać, wystarczy po wykonaniu funkcji sprawdzić, czy na zwróconym przez nią indeksie faktycznie znajduje się liczba n.
