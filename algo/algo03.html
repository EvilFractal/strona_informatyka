sito Eratostenesa

Sito Eratostenesa pozwala szybko określić pierwszość każdej z liczb w przedziale <1, kon>. Aby to osiągnąć, należy utworzyć listę przechowującą informację o pierwszości liczb z tego przedziału, a następnie, idąc od liczb najmniejszych, dla każdej znalezionej liczby pierwszej odznaczyć wszystkie jej wielokrotności (funkcja sito(kon) przypisuje liczbom pierwszym wartość 0, a innym 1). 

vector<bool> sito(int kon){
    vector<bool> sito(kon+1);
    sito[0]=1;
    sito[1]=1;
    for(int i=4;i<=kon;i+=2){
        sito[i]=1;
    }
    for(int i=3;i<kon;i++){
        if(!sito[i]){
            for(int j=i*i;j<=kon;j+=2*i){
                sito[j]=1;
            }
        }
    }
    return sito;
}

Istotnym usprawnieniem jest wstępne odznaczenie wszystkich liczb parzystych za wyjątkiem 2, co pozwala pominąć w następnej pętli liczby parzyste całkowicie. Na uwagę zasługuje także rozpoczynanie odznaczania wielokrotności od kwadratu rozpatrywanej liczby pierwszej - jest to możliwe, ponieważ wszystkie pomijane w tym obrocie pętli liczby zostały odznaczone w jej wcześniejszych obrotach, kiedy nie były od kwadratu rozpatrywanej wtedy liczby większe.
