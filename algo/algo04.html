algorytm Euklidesa

Algorytm Euklidesa służy do szybkiego znajdowania największego wspólnego dzielnika dwóch liczb całkowitych. Funkcja nwd(a,b) korzysta w tym celu z operacji modulo, dopóki jedna z liczb się nie wyzeruje - z drugiej można wtedy odczytać ich największy wspólny dzielnik. 

int nwd(int a,int b){
    int t;
    while(b){
        t=a%b;
        a=b;
        b=t;
    }
    return a;
}
