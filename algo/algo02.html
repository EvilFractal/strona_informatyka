sprawdzanie pierwszości

Sprawdzić, czy liczba naturalna n jest pierwsza oznacza sprawdzić, czy dzieli się ona przez jakąkolwiek inną liczbę naturalną poza 1 i nią samą. Każda liczba złożona posiada co najmniej jeden dzielnik nierówny 1 i jednocześnie nie większy niż jej pierwiastek kwadratowy, zatem aby stwierdzić pierwszość liczby n wystarczy sprawdzić, czy podzielna jest ona przez jakąkolwiek liczbę z tego przedziału.
Funkcja pierwsza(n) na początku sprawdza parzystość liczby n, ponieważ pozwala to później ograniczyć sprawdzanie podzielności tylko do liczb nieparzystych. 

bool pierwsza(int n){
    if(n%2==0 and n!=2){
        return 0;
    }
    for(int i=3;i*i<=n;i+=2){
        if(n%i==0){
            return 0;
        }
    }
    return 1;
}

Dla wielu przypadków (konieczności szybkiego sprawdzenia pierwszości wielu liczb n) metoda ta jest jednak czasochłonna - w takich przypadkach do sprawdzania pierwszości używa się sita Eratostenesa.
